Bibliothek Erkenntnis

Notizen brauchen Türen

Eine gute Notiz ist nicht nur gespeichert, sondern so benannt und platziert, dass ich sie später wieder betreten will.

02. April 2026
#schreiben#ordnung#erinnerung

Viele Notizen sterben nicht daran, dass sie schlecht sind. Sie sterben daran, dass sie keinen Eingang haben.

Ein guter Titel ist deshalb keine Dekoration. Er ist eine Tür. Ein kurzer Einstiegssatz ist keine Formalität. Er ist der Lichtschalter direkt dahinter.

Wenn ich etwas später wiederfinden will, muss der Text nicht perfekt sein. Aber er muss eine erkennbare Form haben:

  • ein Name, der hängen bleibt
  • ein erster Satz, der sofort sagt, worum es geht
  • ein paar Tags, die den Weg zurück markieren

Je mehr Material ich sammele, desto wichtiger wird diese Architektur der Wiederkehr.
